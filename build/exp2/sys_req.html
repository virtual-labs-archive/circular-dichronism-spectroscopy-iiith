To run the source code user system should fulfil the following requirements.

You need to have Python and Visual Python installed on your computer.

To Install Python:

  1. Visit the page http://www.python.org/download/

  2. Download your preffered version (2.5 or above) according to your processor and OS.

  3. For Windows users:
    4. i.Save the downloaded installer file in your preferred directory.
    ii.Go to that directory and double click on the installer file or right click on the file and select Install option.
    iii.A window will open asking for your user preference (default is "for all users"). You can go ahead with the default option or give your own preference and then click next.
    iv.Then it will ask for the location of the folder where you want to save your Python package (default is C:\Python-version). You can go ahead with the default option or give your own preference and then click next.
    v.Then another window will appear asking for which features you want to install (default is "all features"). We recommend you to go ahead with the default option.
    vi.The Python package will now start installing. It will take some time.
    vii.Once it is done click the Finish option.

  4. For Linux users:
    5. Any preffered Python version may be installed through apt-get. (eg for Python version 2.6 you can type $sudo apt-get install python2.6)

To Install Visual Python:

  1. Visit the page http://www.vpython.org/

  2. Select the download option according to your OS.

  3. For Windows users:
    4. i.Download the installer file depending on the Python version available in your system (eg for Python2.7 you have to download VPython-Win-Py2.7-5.41).
    ii.Save the downloaded installer file in your preferred directory.
    iii.Go to that directory and double click on the installer file or right click on the file and select Run option.
    iv.A window will appear asking whether you want to run it or not. Select Run.
    v.Next window is just a welcome page. Click Next.
    vi.Now it will ask "which components should be installed?" We will recommend to go ahead with the default option (ie "full installation") by clicking Next.
    vii.In the next window it will ask whether you want a desktop icon for VPython. If you do not want then uncheck the check box. Otherwise just click Next.
    viii.Now it will start installing. It will take some time.
    ix.Once it is done click the Finish option.

  4. For Linux users:
    5. i.Download and save the visual-5.4_release.tar.bz2 file.
    ii.Extract it at your preffered location.
    iii.Now change change your working directory to that specific location and run the following commands:
    $ ./configure
    $ make
    $ sudo make install
For any error specific to your linux version please refer to the INSTALL.txt file.

If that requirement is fulfilled, you can just run the source code as a script, using the Python interpreter.
