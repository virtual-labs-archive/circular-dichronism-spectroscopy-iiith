1 Flash Plugin

Adobe flash player is the only application that allows you to browse multimedia content like animations, flash games, videos, advertisements, rich media content etc. Adobe flash player is designed to support all the operating systems like windows, MAC, linux, solaris, android etc.

1.1 For Linux

1.1.1 GUI ( Graphical User Interface ) Installation

  • Auto Installation
    • Go to Ubuntu software center. Type Adobe flash plugin in the search box ( Top right ). You can see the Adobe flash plugin in the list.
    • Just press install and wait for the installation to complete.
    • After installation you can view multimedia content in your browser.
  • Manual Installation
    • Go to http://get.adobe.com/flashplayer/ and select the version of flash player suitable for your system to download. This will download a tar.gz plugin in your Downloads folder. b. Unpack the plugin tar.gz and copy the files to the appropriate location. c. Once unpacked, you will get a directory named usr and a file named libflashplayer.so .
    • Open terminal, switch to root account and copy this libflashplayer.so file to /usr/lib/mozilla/plugins. 
      sudo cp /home/username/pathTo/libflashplayer.so /usr/lib/mozilla/plugins
    • Restart the browser and flash will be activated for mozilla.

      For other browsers, identify the location to put plugins and copy this libflashplayer.so file to that location.

1.1.2 CLI ( Command Line Interface ) Installation

  • Before starting the installation, the repository needs to be updated: 
    sudo apt-get update
  • After updating the repository, you can install Adobe flash player in a single command: 
    sudo apt-get install flashplugin-installer
  • All in one command ( Alternative ) Alternatively the following command will install all the multimedia plugins and libraries that needed to play mp3 and video formats. This command will install adobe flash player too: 
    sudo apt-get install ubuntu-restricted-extras

1.2 For Window

  • Go to http://get.adobe.com/flashplayer/ and select the version of flash player suitable for your system. The Flash Player installer should prompt to begin installation after the download is complete. If it does not, start the installer directly from Downloads folder.
  • After downloading Flash Player, close all open browsers.
  • Locate the install flash player file. Unless you have specified otherwise, the file is saved in your Downloads folder.
  • Double-click the installer file to begin the Flash Player installation. This will install flash on your system.

1.3 Testing installation

Go to http://www.adobe.com/software/flash/about/ in your browser and you can see the flash player version details.

1.4 Browser Settings

Even after the installation if the flash does not works

  • Open the menu on top-right corner of the browser.
  • Choose Add-ons.
  • Click the plugins tab.
  • Click on the drop down next to the flash plugin you wish to enable.
  • Select "Ask to Activate" or "Always Activate" from the available choices.
  • Resfresh the pages.
